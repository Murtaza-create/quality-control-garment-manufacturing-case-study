Statistical Quality Control Case Study in Garment Manufacturing
Project Overview

This project presents an industrial case study on improving product quality in a garment manufacturing facility through the application of Statistical Quality Control (SQC) tools. The study focuses on identifying quality defects, analyzing root causes, and proposing process improvements to reduce product rejection and improve manufacturing performance.

Objectives
Analyze quality issues in shirt manufacturing.
Apply Statistical Quality Control tools.
Identify major process defects.
Perform root cause analysis.
Improve process capability and product quality.
Problem Statement

The garment factory experienced a significant number of defective shirts due to process variations and inadequate quality control practices. This project investigates the causes of these defects and demonstrates how statistical quality tools can improve production quality.
